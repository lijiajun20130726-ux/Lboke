打造你的
专属博客

Lboke 是一个简洁优雅的全栈博客系统，基于 Vue 3 + Express + MySQL 构建， 支持文章管理、评论系统、订阅功能，让每个人都能轻松拥有自己的博客。

🚀 访问博客 ⭐ GitHub

功能特点

开箱即用，功能完备

🎨

前后台分离

Vue 3 前端 + Express 后端 API，分层架构清晰，便于维护和扩展

📝

Markdown 编辑

支持 Markdown 编写文章，配合实时预览，让写作更加高效

💬

评论系统

支持游客评论和回复，管理员可审核管理，营造良好的交流氛围

📧

邮件订阅

用户可订阅博客更新，新文章发布时自动通知，保持用户粘性

🔗

友链管理

开放的友链申请系统，自助提交审核，方便博主之间互帮互助

📊

数据统计

管理员后台实时查看网站数据，了解博客运营状况
