# Admin 页面覆盖规则

## 导航
- 侧栏宽度 240px，固定；活动项主题色底
- 图标统一 24×24 SVG，文本字号 15px

## 页面框架
- 外层内边距 32px；移动端 20–24px
- 区块卡片边框 #e2e8f0，圆角 20–24px

## 表单
- 输入圆角 10–12px；聚焦描边主题色
- 操作区右对齐；主按钮主题色、次按钮灰色描边

## 表格
- 表头背景 #f8fafc；行悬停浅主题色
- 状态以颜色标签体现：info/warn/success/error

## 提示与反馈
- 顶部居中 Toast，3s 自动消失；风格与前台一致

## 反模式覆盖
- 移除 Emoji 图标，改用统一 SVG
- 悬停不引起布局偏移
